Version History

Version 1.6

May 17, 2026

This release focuses on stability and reliability. We've added crash reporting so we can detect and resolve issues faster, and strengthened how your history and favorites sync across devices.

  • Crash Reporting: Integrated Firebase Crashlytics for faster diagnosis and resolution of stability issues. Crash data handling is declared in our App Privacy manifest.
  • Reliable Cloud Sync: Improved coordination for History and Favorites sync — your local data is now preserved if an initial cloud fetch fails due to a flaky connection.
  • Under-the-Hood Refactor: Reworked how detection thumbnails are stored on disk for better long-term reliability.

Version 1.5

May 2, 2026

We've upgraded the object detection engine again — this time to YOLOX, a permissively-licensed model that keeps the same 80 detectable objects with an even smaller footprint.

  • New Detection Engine (YOLOX-M): Replaced YOLOv7 with YOLOX-M (Apache 2.0 license). Same recognition coverage, cleaner licensing for App Store distribution.
  • Smaller Model Size: Detection model file reduced by roughly 31% compared to the prior release.
  • Bounding Box Accuracy: Fixed box alignment for portrait-orientation captures so detected objects are framed correctly on screen.

Version 1.4

April 26, 2026

We've revamped the AI-powered Hadith search backend and modernized the app for the latest iOS.

  • New AI Backend: Migrated Hadith search from DigitalOcean GenAI to OpenAI for higher-quality and faster responses.
  • Localized AI Output: AI-generated Hadith summaries now respect your device language.
  • English Narrator Field: Hadith results now include English narrator information alongside the Arabic chain.
  • iOS 18.6 & Modern UI: Bumped the deployment target to iOS 18.6 and adopted refreshed SwiftUI styling APIs.

Version 1.3

June 13, 2025

This release brings push notifications, phone verification, and seamless cross-device sync for your favorites and history.

  • Push Notifications: Stay informed with opt-in alerts. We've integrated Apple Push Notification service and Firebase Cloud Messaging.
  • Phone Number Verification: Optionally verify your phone number with an SMS code via Firebase Phone Auth.
  • Cloud Sync for Favorites & History: With your consent, your favorites and detection history sync securely across your devices.
  • Data Personalization Toggle: New explicit toggle in Account Settings to control consent for personalized content.
  • Expanded Country Codes: Comprehensive alphabetized list of all countries for phone code selection.

Version 1.2

June 9, 2025

Get ready for a smarter, faster, and lighter detection experience! We've supercharged Lightio with the latest advancements in object detection technology.

  • Upgraded Detection Engine: We've replaced our underlying object detection model with YOLOv7, one of the most powerful and efficient real-time object detectors available! This means you can expect:
    • Improved Accuracy: Better identification of objects in your images.
    • Enhanced Performance: Potentially faster detection speeds.
    • State-of-the-Art Technology: Leveraging the cutting edge in AI research to bring you the best possible results.
  • Significantly Smaller App Size: Thanks to this model upgrade and optimizations, Lightio is now much lighter! The app's download size has been reduced by approximately half, saving you storage space.
  • Under-the-Hood Optimizations: We've also made several internal tweaks to ensure smoother operation and reliability.

We're committed to continuously improving Lightio to provide you with the best object detection tool. Update now and experience the difference! We'd love to hear your feedback.
